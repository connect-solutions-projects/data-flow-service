# Operações, Instalação e Testes

Materiais de apoio para preparar o ambiente, executar a stack e validar o sistema.

## Guias essenciais

- `manual-instalacao-implantacao-testes.md` — passo a passo oficial (Docker Compose, certificados, proxy, testes completos).
- `tutoriais/endpoints-dataflow.md` — uso dos endpoints, incluindo o batch `gera-parametros`.
- `tutoriais/grafana-prometheus.md` — guia completo de Grafana e Prometheus (métricas, dashboards, consultas PromQL).
- `tutoriais/scripts-overview.md` — catálogo dos scripts organizados.
- `tutoriais/run-outside-docker.md` — requisitos adicionais caso deseje executar os serviços manualmente (fora do Docker).
- `tutoriais/adicionar-openssl-path.md` — como adicionar OpenSSL ao PATH do PowerShell.

## Tutoriais e utilitários
