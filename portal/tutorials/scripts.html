Scripts Úteis do Projeto

Os utilitários foram organizados em subpastas dentro de scripts/ conforme a finalidade. Utilize este guia para encontrar rapidamente o script certo.

scripts/certs/

  • generate-dev-certs.ps1: gera certificados autoassinados para api.local e reporting.local, produzindo fullchain.pem e privkey.pem.
  • generate-dev-certs.cmd: wrapper para execução via Prompt de Comando.
scripts\certs\generate-dev-certs.cmd

scripts/ingestion/

  • gera-parametros.bat: calcula checksum, tamanho, nome e contentType para uso no POST /ingestion/jobs.
scripts\ingestion\gera-parametros.bat

Informe o caminho completo do arquivo quando solicitado (ex.: C:\dados\lote.csv).

  • gerar-csv-grande.ps1/cmd: gera arquivo CSV de teste com tamanho configurável (padrão 50 MB) e grava em files/large-test-data.csv.
# Interativo
scripts\ingestion\gerar-csv-grande.cmd

# Definindo tamanho
scripts\ingestion\gerar-csv-grande.cmd -TamanhoMB 100

scripts/diagrams/

  • render_mermaid_diagram.ps1: converte o diagrama Mermaid em PNG/SVG usando @mermaid-js/mermaid-cli.
  • render_mermaid_diagram.cmd: wrapper para execução via CMD.
scripts\diagrams\render_mermaid_diagram.cmd

Pré-requisito: Node.js com @mermaid-js/mermaid-cli disponível via npx.

Resumo

  • Certificados → scripts/certs/.
  • Auxílio de ingestão → scripts/ingestion/.
  • Diagramas → scripts/diagrams/.

Mantenha este guia atualizado sempre que novos utilitários forem adicionados ou alterados.
